递归实现指数型枚举

原题链接：https://www.acwing.com/problem/content/94/

从 1∼n 这 n 个整数中随机选取任意多个，输出所有可能的选择方案。
输入格式
输入一个整数 n。
输出格式
每行输出一种方案。
同一行内的数必须升序排列，相邻两个数用恰好 1 个空格隔开。
对于没有选任何数的方案，输出空行。
本题有自定义校验器（SPJ），各行（不同方案）之间的顺序任意。
数据范围
1≤n≤15
**输入样例：**3
输出样例：
3
2
2 3
1
1 3
1 2
1 2 3

算法内容：DFS

前期不熟练，建议画出递归求解树分析
以样例为例画图

ac代码

#include<bits/stdc++.h>
using namespace std;
int st[16]={0};//每个空的状态，0表示还没考虑，1表示选，2表示不选 
int n;
int dfs(int u)
{	
	if(u>n)//判断边界,输出结果 
	{
		for(int i=1;i<=n;i++)
			if(st[i]==1)
				printf("%d ",i);
		printf("\n");
		return 0;		
	}
	dfs(u+1);//不选的分支 
	//st[u]=0 回溯，使其为还没考虑的状态 
	st[u]=1;
	dfs(u+1);//选的分支 
	st[u]=0; 	 	 
}

int main()
{
	while(scanf("%d",&n)!=EOF)
	{	
		dfs(1);	
	}return 0;
		
}
