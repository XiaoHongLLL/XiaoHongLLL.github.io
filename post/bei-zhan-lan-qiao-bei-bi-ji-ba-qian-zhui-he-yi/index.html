备战蓝桥杯笔记（八）—前缀和（一）

2022-01-20

介绍前缀和的模板，原理以及应用

模板

1. 一维前缀和

S[i] = a[1] + a[2] + ... a[i]
[l,r]区间的和 ，即（a[l] + ... + a[r]） = S[r] - S[l - 1];

2. 二维前缀和

S[i][ j] = 第i行j列格子左上部分所有元素的和
S[i][j]=S[i-1][j]+S[i][j-1]+a[i][j]-S[i-1][j-1];


以 (x1,y1) 为左上角， (x2,y2) 为右下角的子矩阵的和为：
S[x2, y2] - S[x1 - 1, y2] - S[x2, y1 - 1] + S[x1 - 1, y1 - 1]];

例题

1. 前缀和
原题链接
题意：
输入一个长度为n的整数序列，输出第L个数到R个数的和

样例

输入：

5 3
2 1 3 6 4
1 2
1 3
2 4

输出：
3
6
10

代码

直接代一维前缀和模板

#include<bits/stdc++.h>
using namespace std;
int n,m,l,r,a[100005],s[100005]; 
int main()
{	
	while(scanf("%d %d",&n,&m)!=EOF)
	{
		for(int i=1;i<=n;i++)
		{
			scanf("%d",&a[i]);
			s[i]+=s[i-1]+a[i];
		}
		while(m--)
		{
			scanf("%d %d",&l,&r);
			int t=s[r]-s[l-1];
			printf("%d\n",t);
		}
	}
	
 }

2. 子矩阵的和
原题链接
题意：
输入一个n行m列的整数矩阵，输出以 x1,y1,x2,y2，为左上角坐标和右下角坐标的子矩阵的和

样例

输入：

3 4 3
1 7 2 4
3 6 2 8
2 1 2 3
1 1 2 2
2 1 3 4
1 3 3 4

输出：
17
27
21

代码

直接代二维前缀和模板

#include<bits/stdc++.h>
using namespace std;
int n,m,q,a[1005][1005],s[1005][1005];

int main()
{
	scanf("%d %d %d",&n, &m, &q);
	for(int i = 1; i <= n; i++ )
		for( int j =1 ; j <= m ; j++ )
			{
				scanf("%d",&a[i][j]);
				s[i][j] = a[i][j] + s[i-1][j] + s[i][j-1] - s[i-1][j-1];
			}
		while(q--)
		{	int x1,x2,y1,y2;
			scanf("%d %d %d %d",&x1, &y1, &x2, &y2);
			int t= s[x2][y2] - s[x1-1][y2] - s[x2][y1-1] + s[x1-1][y1-1];
			printf("%d\n",t);
			
		}return 0;	
 }
